Portfolio

A collection of technical builds, projects, and operational tools from the ni0ntech arsenal.

ThreatHunterX

A Python-powered threat intelligence tool designed to enrich, score, and cache IOCs (Indicators of Compromise) using real-time VirusTotal data. Featuring a Rich CLI interface, local SQLite caching, IOC expiration logic, and a force-refresh override, ThreatHunterX helps analysts make faster, smarter decisions without wasting API calls.

InfoSec Python SQLite APIs

Cloud Sentinel

A serverless threat detection engine built on AWS. Cloud Sentinel processes CloudTrail logs, extracts behavior patterns with Python, and uses machine learning to flag anomalies in real-time. From log ingestion to alerting, this project showcases cloud-native security engineering and ML in action.

AWS Python Lambda SageMaker

SecurityScraper

Security Scraper is a modular web scraping engine built for collecting cyber threat intel from top infosec sources like The Hacker News and BleepingComputer. Designed for cloud automation, it supports JavaScript rendering with headless Chrome, modular spider design, AWS Lambda deployment, and output formats ideal for ML analysis and intel enrichment.

Python Scrapy Lambda Threat Intelligence
